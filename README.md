# Portfólio Pessoal

Este projeto é um portfólio online pessoal desenvolvido com **HTML5**, **CSS3** e **JavaScript puro**, sem a utilização de frameworks ou bibliotecas externas.

O objetivo do site é apresentar minha trajetória, formação, habilidades, projetos desenvolvidos e formas de contato, funcionando como um currículo online.

## Sobre o projeto

O portfólio foi criado como parte dos meus estudos em desenvolvimento web, seguindo uma proposta de site pessoal funcional, responsivo e publicado em um serviço gratuito de hospedagem.

O site apresenta uma interface moderna, com tema escuro como padrão, opção de alternância para tema claro, animações simples, cards de projetos e formulário de contato com validação em JavaScript.

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- Git
- GitHub
- GitHub Pages

## Seções do site

O site contém as seguintes seções:

- **Início:** apresentação principal com foto, resumo profissional e links.
- **Sobre mim:** informações sobre minha trajetória, transição de carreira e interesses.
- **Formação:** formação acadêmica, bootcamp e idiomas.
- **Habilidades:** tecnologias, ferramentas e competências.
- **Portfólio:** projetos desenvolvidos durante meus estudos.
- **Contato:** formulário com validação em JavaScript.

## Funcionalidades

- Menu de navegação entre as seções.
- Layout responsivo para diferentes tamanhos de tela.
- Tema escuro como padrão.
- Alternância entre tema escuro e tema claro.
- Animações ao carregar e ao rolar a página.
- Cards com imagens dos projetos.
- Tags com as tecnologias utilizadas em cada projeto.
- Formulário de contato com validação de campos.
- Simulação de envio de mensagem.

## Portfólio

A seção de portfólio reúne projetos desenvolvidos durante meus estudos e práticas em desenvolvimento web.

Cada card de projeto apresenta:

- imagem representativa;
- nome do projeto;
- breve descrição;
- tecnologias utilizadas;
- link para o repositório no GitHub.

Essa seção poderá ser atualizada futuramente com novos projetos conforme minha evolução na área de tecnologia.

## Responsividade

O site foi desenvolvido para funcionar em diferentes dispositivos, como computadores, tablets e celulares.

A adaptação do layout foi feita com **media queries** em CSS, permitindo melhor visualização em telas menores.
